🚀 주요 기능
포커스 모드 (강추 👍)  
키보드 ↓ 화살표를 누르면 현재 읽는 줄만 또렷하게 보이고 나머지는 흐려집니다. 마치 책에 자를 대고 읽는 듯 집중이 잘 됩니다.

빠른 읽기 지원  
단어 앞부분을 굵게 표시해 눈이 글자를 더 빨리 인식하도록 돕습니다. 포커스 모드와 함께 쓰면 읽는 속도가 확실히 빨라집니다.

마크다운 친화적

코드 블록은 D2Coding 폰트와 함께 자동으로 구문 강조 적용

==형광펜== 문법으로 중요한 부분을 표시 가능

편안한 테마와 폰트

기본은 따뜻한 ‘미색 노트’ 테마

다크 모드 지원

프리텐다드, 본명조, 고운돋움 등 가독성 좋은 폰트 선택 가능

💡 사용 방법
reader.html 파일을 메모장에 붙여넣고 .html 확장자로 저장

크롬이나 엣지 브라우저에 드래그해서 열기

읽고 싶은 글을 붙여넣고 Ctrl + Enter → 몰입 모드 시작
