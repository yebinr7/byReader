# ByReader

몰입형 독서 리더기 — 바이오닉 리딩 · TTS · 포커스 모드 · PWA

🔗 **[바로 사용하기](https://yebinr7.github.io/byReader/ByReader.html)**

---

## 사용 방법

**로컬 리더기 (ByReader.html)**
1. `ByReader.html`을 Chrome/Edge로 열기
2. 텍스트 또는 마크다운 붙여넣기
3. `Ctrl + Enter` → 읽기 시작

**크롬 확장프로그램**
1. Chrome → `chrome://extensions` → 개발자 모드 ON
2. "압축 해제된 확장 프로그램 로드" → 이 폴더 선택
3. 아무 웹페이지에서 확장 아이콘 클릭 → 리더 모드 ON/OFF

**모바일 (PWA)**
1. 폰 브라우저에서 [링크](https://yebinr7.github.io/byReader/ByReader.html) 접속
2. "홈 화면에 추가" 배너 클릭
3. 홈화면 아이콘으로 앱처럼 실행

## 주요 기능

| 기능 | 설명 |
|------|------|
| **마크다운 렌더링** | `#` 제목, `**굵게**`, `` `코드` ``, `==형광펜==`, 코드블록 |
| **바이오닉 리딩** | 단어 앞 40~50%를 굵게 → 눈이 빠르게 인식 |
| **포커스 모드** | 현재 줄만 선명, 나머지 흐리게 → ↑↓ 키로 이동 |
| **TTS** | 텍스트 음성 읽기 · 목소리/배속 선택 · 진행 바 · 10초 뒤로 |
| **백그라운드 TTS** | 화면 꺼져도 계속 읽음 (런닝 중 사용 가능) |
| **잠금화면 컨트롤** | 알림바에서 재생/일시정지/10초 뒤로 |
| **테마** | 라이트 / 미색 노트 / 다크 모드 |
| **폰트** | 프리텐다드 · 본고딕 · 본명조 · 고운 돋움 · D2Coding |
| **설정 저장** | 폰트·크기·행간·테마 자동 기억 |
| **오프라인 동작** | Service Worker 캐시 → 인터넷 없어도 작동 |

## 백그라운드 TTS 원리

일반 브라우저는 화면이 꺼지면 JS 실행을 멈춥니다.
ByReader는 세 가지 방법으로 이를 우회합니다:

1. **AudioContext 무음 루프** — 오디오 세션을 점유해서 OS가 앱을 백그라운드로 중단시키지 않게 함
2. **Media Session API** — OS에 "음악 재생 중"으로 등록 → 잠금화면에 컨트롤 표시
3. **Chrome 장문 버그 우회** — 10초마다 pause/resume으로 15초 멈춤 현상 방지

## 파일 구조

```
ByReader/
├── ByReader.html      # 로컬 리더기 (PWA)
├── app.webmanifest    # PWA 앱 정보 (이름, 아이콘, 테마색)
├── sw.js              # Service Worker (오프라인 캐시)
├── icon.svg           # 앱 아이콘
├── manifest.json      # 크롬 확장프로그램 설정
├── background.js      # 확장프로그램 아이콘 클릭 핸들러
├── content.js         # 웹페이지에 주입되는 리더 UI
└── style.css          # 확장프로그램 스타일
```
